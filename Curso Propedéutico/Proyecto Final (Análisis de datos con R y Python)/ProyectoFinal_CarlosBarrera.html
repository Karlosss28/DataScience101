def seudoinversa(X):
    'Calcula la seudoinversa de la matriz X'
    U, S, VT = np.linalg.svd(np.dot(X.T,X))
    SeudoS = sigmainversa(S, VT.shape[0])
    return np.dot(VT.T, np.dot(SeudoS, U.T))

def sigmainversa (s, m):
    'Regresa la matriz inversa de sigma truncando a cero valores demasiado pequeños'
    S = np.zeros([m, m])
    for idx, val in enumerate(s):
        if (val > 1e-12 and idx < m):
            S[idx][idx] = 1.0/val
    return S

def betafinal(X,Y):
    'Regresa el vector Beta obtenido de la multiplicación de (XT*X)^-1*XT*Y'
    return np.dot(seudoinversa(X), np.dot(X.T,Y))

Cálculo de la bondad de ajuste $R^2$¶

Calculemos ahora el coeficiente de determinación $R^2$ para poder estimar el error asociado a la aproximación lineal que calculamos.

In [14]:
def bondad_ajuste(Y, Yaprox):
    Ymedia = np.mean(Y)
    'Calcula el coeficiente de determinación'
    SEC = 0
    STC = 0
    for yi in Y:
        STC += (np.float_(yi)-Ymedia)**2
    for yaproxi in Yaprox:
        SEC += (np.float_(yaproxi)-Ymedia)**2
    R2 = SEC/STC
    return R2

Interpretación del $R^2$¶

El valor de $R^2$ es de aproximadamente 0.64, este número es el coseno del ángulo existente entre el vector de la variable que pretendemos explicar W, al calcular el arccos de dicho valor, encontramos que el ángulo entre ambos vectores es de 50 grados.

Lo anterior significa que aproximadamente un 64% de la información del vector de la variable explicada está siendo representado por la aproximación, esto nos hace ver que la aproximación es algo pobre ya que pierde casi el 36% de la información. Mientras el ángulo entre los dos vectores se acerca a cero, la información está siendo mejor representada.

Conclusiones¶

Como pudimos observar, los datos meramente ofensivos como lo son el porcentaje de Tiros de Campo, de Tiros de 3 Puntos, de Tiros Libres y el total de puntos, no pueden predecir adecuadamente el número de partidos que un equipo ganará en una temporada regular. Esta situación es explicable, pues también deben tomarse en consideración otro tipo de variables, particularmente, las variables defensivas como lo pueden ser los robos de balón y los bloqueos.
